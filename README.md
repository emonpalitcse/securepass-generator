# 🔐 SecurePass - Password Generator

A clean, responsive, mobile-friendly password generator web app built using HTML, CSS, and JavaScript. Supports:

- Customizable length & character options
- Copy to clipboard
- Password strength indicator
- Dark mode toggle 🌙

- Reset/Clear functionality

➡️ [Try it live](https://emonpalitcse.github.io/securepass-generator/)
